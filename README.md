# flappy-bird
A browser-based Flappy Bird clone built with pure HTML5 Canvas and JavaScript. No dependencies, no frameworks — a single index.html file.

Features:

Smooth 60fps game loop
Parallax background with stars, clouds and ground
Animated bird with wing flapping and rotation
Particle effects on flap and death
Score tracking with personal best
Works on desktop and mobile (tap to flap)
Controls: Space / Click / Tap
